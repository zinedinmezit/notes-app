# notes-app

## Description
Note-taking type of Android application. Besides requiring information for the note to be created, the user can select a color and define priority. This helps the user to sort notes in various ways. Notes can also have a reminder. When the reminder is triggered, the application sends a notification with a descriptive message. All notes that have a reminder are pinned on the calendar.

## Features
* CRUD notes
* Search notes
* Sort notes
* Set remainder
* Calendar implementation

## Tools and libraries
* MVVM architecture
* Navigation component
* Data binding
* Room
* Kotlin Coroutines
* Material design

* *Afollestad* material dialogs (https://github.com/afollestad/material-dialogs) 
* *kizitonwose* CalendarView (https://github.com/kizitonwose/CalendarView)
